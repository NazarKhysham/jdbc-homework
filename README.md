# JDBC Homework

Навчальний проєкт з використання JDBC та бази даних H2.

Метою роботи є отримання практичних навичок роботи з базою даних через Java: створення з'єднання, виконання SQL-запитів, заповнення таблиць даними та отримання результатів у вигляді Java-об'єктів.

## Функціонал

- Підключення до бази даних H2 через JDBC
- Ініціалізація структури бази даних з SQL-файлу
- Заповнення таблиць тестовими даними
- Виконання SQL SELECT-запитів
- Використання PreparedStatement для роботи з SQL
- Отримання результатів у вигляді Java-об'єктів

## Структура проєкту

```text
sql/
src/main/java/com/goit/jdbc/
```

Основні класи:

- `Database` — створення та зберігання підключення до БД
- `DatabaseInitService` — створення таблиць
- `DatabasePopulateService` — наповнення таблиць даними
- `DatabaseQueryService` — виконання SQL-запитів
- `Main` — демонстрація роботи програми

## Запуск проєкту

1. Запустити `DatabaseInitService`
2. Запустити `DatabasePopulateService`
3. Запустити `Main`

## Технології
